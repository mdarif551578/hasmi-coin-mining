# Hasmi Hustle

Welcome to Hasmi Hustle, a gamified token economy application built with Next.js in Firebase Studio. This project provides a feature-rich platform for users to mine, trade, and manage virtual assets.

## Core Features

-   **Authentication**: Secure user authentication with email/password and social providers like Google. Includes signup, login, and password reset flows.
-   **Dual-Wallet System**: A clear separation between a user's deposited **USD** balance and their earned **Hasmi Coin (HC)** balance.
-   **Mining Center**: Users can participate in free daily HC claims or subscribe to paid plans for higher mining rates.
-   **P2P Marketplace**: A platform for users to create offers and trade their HC with other users.
-   **USD to HC Exchange**: A one-way system for users to convert their USD balance into HC at a fixed rate.
-   **Tasks & Earn**: Users can complete simple tasks to earn additional HC rewards.
-   **Referral System**: Users can share a unique code to refer new members and earn bonus tokens.
-   **Modern UI**: A responsive and consistent design built with ShadCN UI and Tailwind CSS for a professional user experience on both desktop and mobile.

## Getting Started

The application starts with a full authentication flow.

1.  Navigate to `/signup` to create a new account.
2.  Navigate to `/login` to sign in.
3.  Once authenticated, you will be redirected to the main dashboard to access all features.

## Tech Stack

-   **Framework**: [Next.js](https://nextjs.org/) (with App Router)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **UI Components**: [Shadcn/ui](https://ui.shadcn.com/)
-   **Icons**: [Lucide React](https://lucide.dev/guide/packages/lucide-react)
